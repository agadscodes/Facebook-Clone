# Facebook Clone 👥

A responsive front-end replication of the Facebook web interface, built using modern semantic HTML5 and custom CSS3 layout techniques. This project focuses on pixel-perfect UI precision, component structural integrity, and responsive design patterns.

---

## 🚀 Features

* **Authentication Page Layout:** A clean recreation of the Facebook landing/login page featuring the split-screen branding and localized forms.
* **Main News Feed:** A dynamic, multi-column layout featuring the Navigation Sidebar, Main Feed, and Sponsored/Contacts Sidebar.
* **Stories Section:** A horizontally scrollable stories component with hover effects and user profile overlays.
* **Create Post Component:** A replicated interactive input container complete with dummy action buttons (Live Video, Photo/Video, Feeling/Activity).
* **Responsive Architecture:** Fully optimized for fluid viewing across desktop, tablet, and mobile screen sizes using media queries.
* **UI Accuracy:** Custom styled scrollbars, icon placement, typography matching, and official Facebook branding color palettes.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic structuring (`<nav>`, `<main>`, `<aside>`, `<section>`) for clean SEO-friendly and accessible markup.
* **CSS3:** Custom properties (CSS variables) for consistent theme handling, Flexbox, and CSS Grid for advanced layouts.
* **FontAwesome / Google Fonts:** For accurate iconography and clean modern typography.
